# [Advent of code 2020 in Rust](https://github.com/ekroon/adventofcode2020/)

## Running

* Install [Rust](https://www.rust-lang.org/tools/install) 
* Install [Cargo AOC](https://github.com/gobanos/cargo-aoc)
